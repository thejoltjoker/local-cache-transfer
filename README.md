# Local Cache Transfer

This is a simple PyQt5 application that allows you to drag and drop files and folders for copying to a destination folder. It also calculates and displays the total size of the dropped folder's contents.

## Features

- Drag and drop folders for copying.
- Copy progress is shown as a percentage.

## Requirements

- Python 3.x
- PyQt5
